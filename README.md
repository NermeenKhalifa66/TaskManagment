# Task Management Application

A simple full-stack task management app built for a technical assessment.
Users can register, log in, and manage their own tasks.

## Tech Stack
- Node.js (learning)
- Express (learning)
- SQLite
- React (completed)
- JWT Authentication

## About Me

API Endpoints

Auth
- POST /api/auth/register
- POST /api/auth/login

Tasks
- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/:id
- DELETE /api/tasks/:id
